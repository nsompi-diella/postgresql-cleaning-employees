# SQL Data Cleaning EMPLOYEES TABLE ( INTERMEDIATE LEVEL )

## Task

A client provided an employee dataset and asked me to clean and transform the data by completing the following tasks:

1. Convert the **Full Name** to uppercase.
2. Separate the **Phone Number** into **Phone Code** and **Phone Number**.
3. Split the **Email Address** into:

   * Username
   * Domain Name
   * Domain Extension
4. Clean the **Age** column:

   * Remove NULL values.
   * Keep only ages greater than 0.
5. Create **Age Categories** (Infant, Child, Adult, Senior, etc.).
6. Join the **Departments** table and retrieve the **Department Name**.
7. Clean the **Salary** column:

   * Remove NULL values.
   * Keep only salaries greater than 0.
   * Round salary values.
8. Join the **Countries** table and retrieve the **Country Name**.
9. Clean the **Status** column:

   * Remove NULL or empty values.
   * Keep only active employees.
10. Replace NULL values in the **Address** column.
11. Format the **Hire Date** as **DD/MM/YYYY**.
